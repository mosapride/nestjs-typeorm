# NestJS TypeORM

## 実行環境について

3パターンの環境での実行を想定とする。下記は設定する必要はない。

| 環境変数=値        | envファイル名     | 意味                  |
|---------------|--------------|---------------------|
| LEVEL=DEBUG   | .debug.env   | デバックモード。開発時に使用する。   |
| LEVEL=TEST    | .test.env    | テストモード。結合テストなどに使用する |
| LEVEL=RELEASE | .release.env | リリース環境。本番環境にて使用する   |

### 必要設定

DataBaseの設定が必要となる。
